# HEX Color Extractor

一个简洁优雅的图片色彩提取工具，自动识别图片中的主色调、辅色和点缀色。

![Preview](preview.png)

## 功能特点

- **智能提取** - 自动识别图片中占比最高的 5 种颜色
- **分类展示** - 将颜色分为主色调、辅色、点缀色三类
- **多种上传方式**
  - 点击上传
  - 拖拽上传
  - 粘贴上传 (Ctrl+V / Cmd+V)
- **一键复制** - 点击单个颜色复制，或一键复制全部色值
- **纯前端实现** - 无需服务器，本地即可运行

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 上传、拖拽或粘贴图片
3. 等待分析完成，查看提取的颜色
4. 点击颜色卡片或「一键复制全部」按钮获取 HEX 色值

## 技术栈

- HTML5 Canvas (图像处理)
- Vanilla JavaScript (颜色提取算法)
- CSS3 (响应式布局)

## 颜色提取算法

1. 将图片缩放至 150px 以提高处理速度
2. 对像素颜色进行量化 (16 级) 减少相似颜色
3. 统计各颜色出现频率并排序
4. 使用欧氏距离筛选差异明显的颜色
5. 输出占比最高的 5 种不同颜色
